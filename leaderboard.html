🏆 Leaderboard Ranking System

Who's leading the scoreboard? This **C++ program** updates the **leaderboard** dynamically based on new scores! Enter new scores and watch the rankings shift! 🚀

📊 Expected Output

  • 📌 **Previous Leaderboard:**
    🎮 Player 1 - 🎖️ 4
    🎮 Player 2 - 🎖️ 1
    🎮 Player 3 - 🎖️ 2
    🎮 Player 4 - 🎖️ 3
  • 📝 Enter new scores: **5 3 7 1**
  • 🏆 **Updated Leaderboard:**
    🥇 **1st Place** - 🎮 Player 3 - 🎖️ 7
    🥈 **2nd Place** - 🎮 Player 1 - 🎖️ 5
    🥉 **3rd Place** - 🎮 Player 4 - 🎖️ 3
    4️⃣ **4th Place** - 🎮 Player 2 - 🎖️ 1

💻 C++ Code


#include <iostream>

using namespace std;

int main() {
    int scores[] = {4, 1, 2, 3}; 
    int players[] = {1, 2, 3, 4};
    int size = 4;

    cout << "📌 Previous Leaderboard:\n";
    cout << "🎖️ Player  -  Score\n";

    for (int i = 0; i < size; i++) {
        cout << "🎮 " << players[i] << "  -  " << scores[i] << "\n";
    }

    int newScores[size];
    cout << "\n📝 Enter " << size << " new scores: ";
    
    for (int i = 0; i < size; i++) {
        cin >> newScores[i];
    }

    int newPlayers[] = {4, 1, 3, 2};

    for (int i = 0; i < size - 1; i++) {
        for (int j = i + 1; j < size; j++) {
            if (newScores[i] < newScores[j]) {
                swap(newScores[i], newScores[j]);
                swap(newPlayers[i], newPlayers[j]);
            }
        }
    }

    cout << "\n🏆 Updated Leaderboard:\n";
    cout << "🏅 Placement  -  🎮 Player  -  🎖️ Score\n";

    for (int i = 0; i < size; i++) {
        cout << "   " << (i + 1) << "         -   " 
             << newPlayers[i] << "       -   " 
             << newScores[i] << "\n";
    }

    return 0;
}
