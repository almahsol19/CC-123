🔍 Most Frequent Number Finder in C++

This **C++ program** identifies the **most frequently occurring number** in an array. It efficiently **allocates memory**, processes input, and selects the **smallest number** in case of a tie. ✅ Ensures **proper memory management** before exiting.

📌 Expected Output

📊 Enter the number of elements: 7
🔢 Enter 7 numbers: 1 2 1 1 4 5 6
🏆 The most frequent number is: 1

💻 C++ Code


#include <iostream>

int main() {
    int n;

    std::cout << "📊 Enter the number of elements: ";
    std::cin >> n;

    int* arr = new int[n];

    std::cout << "🔢 Enter " << n << " numbers: ";
    for (int* ptr = arr; ptr < arr + n; ++ptr) {
        std::cin >> *ptr;
    }

    bool* counted = new bool[n](); 

    int maxFrequency = 0;
    int mostFrequentNumber = 0;

    for (int i = 0; i < n; ++i) {
        if (counted[i]) continue;

        int count = 1;
        for (int j = i + 1; j < n; ++j) {
            if (arr[i] == arr[j]) {
                count++;
                counted[j] = true;
            }
        }

        if (count > maxFrequency || (count == maxFrequency && arr[i] < mostFrequentNumber)) {
            maxFrequency = count;
            mostFrequentNumber = arr[i];
        }
    }

    std::cout << "🏆 The most frequent number is: " << mostFrequentNumber << std::endl;

    delete[] arr;
    delete[] counted;

    return 0;
}
