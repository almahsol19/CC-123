🎓 Student Grade Calculator in C

This interactive C program **calculates student grades** based on their performance in quizzes, assignments, and exams. It utilizes a **weighted formula** (20% quiz, 30% assignment, 50% exam) to compute the final grade and assigns a **letter grade** accordingly.

📝 Expected Output

📌 Enter the number of students: 2

👩‍🎓 Student 1:
📝 Quiz Score (0-100): 90
📂 Assignment Score (0-100): 86
🏆 Exam Score (0-100): 89
🎯 Final Grade: 88.30
📜 Letter Grade: B

👨‍🎓 Student 2:
📝 Quiz Score (0-100): 90
📂 Assignment Score (0-100): 75
🏆 Exam Score (0-100): 90
🎯 Final Grade: 85.50
📜 Letter Grade: B

💻 Code


#include <stdio.h>

int main() {
    float quiz, assignment, exam, finalGrade;
    char grade;
    int enterNumberOfStudent = 0;

    printf("🎓 Enter the number of students: ");
    scanf("%d", &enterNumberOfStudent);

    for (int student = 0; student != enterNumberOfStudent; student++) {
        printf("\n👩‍🎓 Student %d:\n", student + 1);
        
        printf("📝 Quiz Score (0-100): ");
        scanf("%f", &quiz);
        
        printf("📂 Assignment Score (0-100): ");
        scanf("%f", &assignment);
        
        printf("🏆 Exam Score (0-100): ");
        scanf("%f", &exam);

        finalGrade = (quiz * 0.20) + (assignment * 0.30) + (exam * 0.50);

        if (finalGrade >= 90) {
            grade = 'A';
        } else if (finalGrade >= 80) {
            grade = 'B';
        } else if (finalGrade >= 70) {
            grade = 'C';
        } else if (finalGrade >= 60) {
            grade = 'D';
        } else {
            grade = 'F';
        }

        printf("🎯 Final Grade: %.2f", finalGrade);
        printf("\n📜 Letter Grade: %c\n", grade);
    }

    printf("\n✅ All grades calculated successfully!\n");
    return 0;
}
